## Updating Git

Git for Windows includes a built-in command to update itself to the latest version.

### Using update-git-for-windows

```bash
git update-git-for-windows
```

This command checks for the latest version and updates Git automatically.

**Minimum version:** Git 2.16.1 or later

### Older Versions

If you're running Git 2.14.2 to 2.16.1, use:

```bash
git update
```

For Git versions older than 2.14.1, download and reinstall from [gitforwindows.org](https://gitforwindows.org/).

## Checking Your Git Version

```bash
git --version
```
