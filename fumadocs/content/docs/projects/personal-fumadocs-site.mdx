# Personal Fumadocs Site

Personal documentation and portfolio site built with Fumadocs and Next.js.

## Repository

- **GitHub**: https://github.com/RelativeSure/websites
- **Status**: Active Development

## Tech Stack

- Fumadocs
- Next.js
- TypeScript
- Tailwind CSS

## Getting Started

```bash
git clone https://github.com/RelativeSure/websites
cd websites/fumadocs
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
