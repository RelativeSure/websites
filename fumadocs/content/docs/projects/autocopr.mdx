# AutoCOPR

Automation tools for COPR (Cool Other Package Repo) - automating spec file updates and package building workflows.

## Repository

- **GitHub**: https://github.com/RelativeSure/autocopr
- **Status**: Active Development

## Tech Stack

- Python
- RPM/Spec files
- COPR API
- Shell Scripts

## Overview

AutoCOPR is a collection of tools designed to streamline the process of maintaining packages in Fedora COPR repositories. It automates spec file updates and simplifies the package building workflow.

## Getting Started

```bash
git clone https://github.com/RelativeSure/autocopr
cd autocopr
```
