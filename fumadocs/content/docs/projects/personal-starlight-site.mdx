Documentation site built with Starlight and Astro.

## Repository

- **GitHub**: https://github.com/RelativeSure/websites/tree/master/starlight
- **Status**: Active Development

## Tech Stack

- Starlight
- Astro
- MDX
- TypeScript

## Getting Started

```bash
git clone https://github.com/RelativeSure/websites
cd websites/starlight
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) to view the site.
