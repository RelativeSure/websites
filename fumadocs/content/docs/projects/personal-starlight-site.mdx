# Personal Starlight Site

Documentation site built with Starlight and Astro.

## Repository

- **GitHub**: https://github.com/RelativeSure/personal-starlight-site
- **Status**: Active Development

## Tech Stack

- Starlight
- Astro
- MDX
- TypeScript

## Getting Started

```bash
git clone https://github.com/RelativeSure/personal-starlight-site
cd personal-starlight-site
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) to view the site.
