# Fumadocs Website

This directory is reserved for a Fumadocs-based website.

## Setup

Import or create your Fumadocs project here. Example structure:
```
fumadocs/
├── package.json
├── next.config.js
├── app/
│   └── ...
└── ...
```

## Getting Started

Once you've added your Fumadocs project, you can typically run:

```bash
npm install
npm run dev
```
