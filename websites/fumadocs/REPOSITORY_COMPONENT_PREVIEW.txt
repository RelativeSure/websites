================================================================================
                          FEATURES HIGHLIGHTED
================================================================================

✅ Clickable Repository Names
   - Direct link to GitHub repository
   - Opens in new tab
   - Changes color on hover

✅ Status Badges
   - Automatic color based on status
   - Shows at-a-glance project health
   - Three options: maintained, in-progress, archived

✅ Technology Tags
   - Display skills and tech stack
   - Hash-prefixed for visual clarity
   - Multiple tags per project

✅ "View Repo" Button
   - Prominent call-to-action
   - Primary color button
   - Hover effect for feedback

✅ Responsive Layout
   - Stacks vertically on mobile
   - Multi-column on desktop
   - Maintains readability at all sizes

✅ Dark Mode Compatible
   - Automatic dark mode styling
   - Proper contrast ratios
   - Professional appearance in both modes

================================================================================
                            INTEGRATION POINTS
================================================================================

Used on:
  ✓ Home page - "My Repositories" section at bottom
  ✓ Any MDX documentation page
  ✓ Custom React components

Example Usage:
  import RepositoryBadges from '@/components/RepositoryBadges';

  <RepositoryBadges 
    repositories={[
      {
        name: 'my-project',
        url: 'https://github.com/user/my-project',
        description: 'Amazing project',
        topics: ['typescript', 'react'],
        status: 'maintained',
      },
    ]}
  />
